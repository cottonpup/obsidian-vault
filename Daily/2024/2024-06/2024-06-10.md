# 자료구조
## Ch2 문제풀이
### 01. 다음 수열의 n번째 항을 구하는 재귀 알고리즘을 작성하시오.
```java
public class Sequence {
    public static int seq(int n) {
        if (n == 1) return 0;
        return 5 * seq(n - 1) + 3;
    }

    public static void main(String[] args) {
        int n = 5; // 예시로 5번째 항을 구해봅니다.
        System.out.println("seq(" + n + ") = " + seq(n));
    }
}
```

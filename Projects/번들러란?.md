# 번들러란?
번들러는 웹 리소스들을 최적화하고 묶어주는 도구입니다. 예를 들어, 여러 개의 자바스크립트 파일을 하나로 합치거나, 이미지를 압축하는 작업을 자동으로 해줍니다. 이렇게 해서 웹 페이지의 로딩 속도를 개선하고 코드 관리를 더 효율적으로 할 수 있습니다.

## Emotion 에서 `>` Selector

  

```js

const TextLogo = styled.h1`

font-size: 24px;

font-weight: 700;

> span[class="primary"] {

color: rgb(255, 47, 100)

}

> span:not(.primary) {

color: #222;

}

`;

```

> 는 자식 선택자였다! 위 코드에서는 `h1 > span` 이 되는 것.
